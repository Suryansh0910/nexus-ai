## üöÄ Getting Started

1. **Clone the repo:**
   ```bash
   git clone https://github.com/Suryansh0910/nexus-ai.git
   ```

2. **Backend Setup:**
   - Navigate to `backend/`
   - Run `npm install`
   - Create a `.env` with `MONGO_URI` and `JWT_SECRET`
   - Start with `node index.js`

3. **Frontend Setup:**
   - Navigate to `frontend/`
   - Run `npm install`
   - Create a `.env` with your OpenRouter keys (`VITE_MISTRAL_KEY`, etc.)
   - Start with `npm run dev`
